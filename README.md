## 🚀 How to Use

### Option 1: Use Online (GitHub Pages)
Just open the live link above — no installation needed.

### Option 2: Download & Open Locally
1. Download `index.html`
2. Open it in any modern browser (Chrome, Firefox, Edge, Safari)
3. Works 100% offline — no server needed!

### Option 3: Install as PWA
1. Open in Chrome or Edge
2. Click the install icon in the address bar (or the install banner in the app)
3. Use it like a native app on your phone or desktop

## 🎓 About the Test

The **Einbürgerungstest** (Leben in Deutschland) is the official German naturalization test. It consists of:
- **300 general questions** (same for all of Germany)
- **10 state-specific questions** (different per Bundesland)
- **NRW** (Nordrhein-Westfalen) has its own set of regional questions included here

To pass the real test, you need **17 out of 33 questions** correct (≈ 51%).
